About the Project

This frontend demonstrates a college attendance system UI built with pure HTML, CSS and JavaScript. It uses localStorage to persist attendance and allows export/import in CSV for demo purposes. Use it as a portfolio project on GitHub Pages.

Why this project?

  • Shows UI/UX skills (responsive layout, cards, gradients).
  • Demonstrates JavaScript DOM manipulation and localStorage usage.
  • Easy to extend later with backend (Node/Express) for real data.
